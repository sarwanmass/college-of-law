🔧 METHOD 2: GITHUB + AUTOMATIC UPDATES (PROFESSIONAL - 10 MINUTES)

Why? Auto-deploy every time you push code = zero manual uploading!

STEP 1: Create GitHub Repository
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Go to: https://github.com/new
  2. Repository name: "kuvempu-college" (or any name)
  3. Description: "RASHTRAKAVI KUVEMPU COLLEGE OF LAW Website"
  4. Select: "Public" (so Netlify can access)
  5. Click: "Create repository"

STEP 2: Push Code to GitHub
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  Open Terminal/Command Prompt in the "kuvempu-law-college" folder:

  git init
  git add .
  git commit -m "Initial commit: Kuvempu College website"
  git branch -M main
  git remote add origin https://github.com/YOUR_USERNAME/kuvempu-college.git
  git push -u origin main

  [Replace YOUR_USERNAME with your actual GitHub username]

STEP 3: Connect GitHub to Netlify
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Go to: https://app.netlify.com
  2. Click: "Add new site" → "Import an existing project"
  3. Select: "GitHub"
  4. Authorize Netlify to access GitHub
  5. Select your "kuvempu-college" repository
  6. Settings:
     - Branch to deploy: main
     - Build command: (leave empty)
     - Publish directory: . (dot)
  7. Click: "Deploy site"

⚠️ IMPORTANT: DATA STORAGE & BACKUP

Where is data stored?
  • Browser's localStorage (not on a server)
  • Private to each visitor's device
  • Does NOT sync across devices

What does this mean?
  ✓ Fast & secure
  ✗ Could be lost if browser cache cleared

SOLUTION: Weekly Backups!
  1. Admin Panel → Backup Data
  2. Click "Download Backup"
  3. Save the .json file somewhere safe
  4. Keep in Google Drive / OneDrive / USB

RESTORE FROM BACKUP:
  1. Admin Panel → Backup Data
  2. Click "Restore from Backup"
  3. Select your saved .json file
  4. All data restored! ✓

❓ TROUBLESHOOTING

Problem: "Page not loading" or "404 error"
Solution:
  • Check all files are in Netlify (dashboard → Files tab)
  • Clear browser cache (Ctrl+Shift+Delete)
  • Try incognito mode

Problem: "Photos disappeared after refresh"
Solution:
  • localStorage may be cleared
  • Always keep backups (Admin → Backup)
  • Re-upload photos via Admin panel

Problem: "Site not updating after git push"
Solution:
  • Go to Netlify dashboard → Deploys
  • Click latest deploy to see logs
  • Check for build errors

Problem: "Admin panel links don't work"
Solution:
  • Make sure netlify.toml is in root folder
  • Clear browser cache
  • Try different browser
